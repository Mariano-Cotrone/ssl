# TP0 - Hello World en C

## Autor

Mariano Cotrone

## Compilador seleccionado

GCC

## Versión del compilador

`gcc.exe (Rev2, Built by MSYS2 project) 16.1.0`

## Versión del lenguaje C

C23

## Archivos del trabajo

* `hello.c`: código fuente del programa.
* `output.txt`: archivo con la salida obtenida al ejecutar el programa.
* `Makefile`: archivo utilizado para automatizar la compilación y la ejecución.
* `README.md`: documentación del trabajo práctico.

## Compilación

Para compilar el programa se utiliza:

```bash
make
```

## Ejecución

Para ejecutar el programa se utiliza:

```bash
make run
```

## Salida obtenida

```txt
Hello, World!
```
